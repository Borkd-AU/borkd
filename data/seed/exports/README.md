# Seed data exports

Generated artefacts from seed scripts.

## Files

- `seeded-pins-sydney.geojson` — all permanent pins from Supabase exported
  as GeoJSON. Drag into https://geojson.io to visualise on a map.
  Generated by `pnpm --dir scripts export:geojson`.

Regenerate anytime after re-running `pnpm --dir scripts seed`.
